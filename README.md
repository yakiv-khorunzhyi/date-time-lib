# DateTime
Simple library for working with DateTime.

## Requirements
| master    | 1.0      |
|:----------|:--------:|
| php       | 7.2      |

## Installation
Install the latest version with:
```
$ composer require yakiv-khorunzhyi/date-time-lib
```
